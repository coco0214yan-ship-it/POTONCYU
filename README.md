# ぽとんちゅ 穴あき一首ガチャ

大学のがやがやした日常の中で、友達と空欄入りの一首を引き、4択で完成させる静的プロトタイプです。

## 画面の流れ

1. トップで「まわしてみる」を押す
2. 今の空気に近い台を選ぶ
3. 穴あき一首と4択が出る
4. 選んだ言葉で一首が完成する
5. 一問だけ置かれるので、あとは目の前の友達と話す

## 仕様

- 外部サービスは使いません
- 入力欄はありません
- 見返すための記録は残しません
- 印をつけたり外へ送ったりする機能はありません
- 一首データはJavaScript内の配列で管理しています

## 開き方

サーバーなし:

1. `index.html` をダブルクリック

ローカルサーバー:

1. PowerShellで `C:\Users\Owner\Documents\coco` を開く
2. `node server.js` を実行
3. `http://127.0.0.1:4173/honne-dialogue-mvp/` を開く
